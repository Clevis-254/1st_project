## Installation

To install the project, follow these steps:

```bash
$ git clone https://github.com/Clevis-254/1st_project.git
$ cd 1st_project
$ npm install
```
## Packages
The packages used are python flask and Tailwind css packages.
To install the flask package do the following:
```bash
$ pip install flask.
```
